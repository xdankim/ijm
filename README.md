## What is this?

A file that read the jar mods to copy and import them into Minecraft Java folder `.minecraft/mods`

## How to use it?

1. Double click file you just downloaded and it show the notice about an error to create a folder. just click ok (ignore them)
2. Copy the current path of location folder which all your mods in that folder. let say i save my mods at `C:\Users\{windowname}\Downloads`. so, just copy the location and paste it.
3. Copy your `C:\Users\...\.minecraft\mods` location.
4. Choose your mods to copy and click ok

- Mods located (default) : `C:\Users\{windowname}\Downloads`
- Minecraft Folder : `C:\Users\{windowname}\AppData\Roaming\.minecraft\mods`;
