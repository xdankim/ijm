## IJM (Import Java Mods)

Built a tool that imports Java Minecraft mods directly into the game without needing to open the mods folder to paste the mods.

## How to use it?

1. Double click file you just downloaded and it show the notice about an error to create a folder. just click ok (ignore them)
2. Copy the current path of location folder which all your mods in that folder. let say i save my mods at `C:\Users\{USERPROFILE}\Downloads`. so, just copy the location and paste it.
3. Copy your `C:\Users\...\.minecraft\mods` location.
4. Choose your mods to copy and click ok

- Mods located (default) : `C:\Users\{USERPROFILE}\Downloads`
- Minecraft Folder : `C:\Users\{USERPROFILE}\AppData\Roaming\.minecraft\mods`;

## is this actually useful?

if you are lazy to copy and paste or drag the mods into the mods folder you can use this instead. i made this tool for my friend which he didnt know about computer. You dont need to press `⊞` + `r` to know the location of game. Java already read where the path of legacy launcher stay.
