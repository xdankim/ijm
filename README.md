## What is this?

A file that read the jar mods to copy and import them into Minecraft Java folder ".minecraft/mods"

## How to use it?

1. Double click that file and it show the notice about to create a folder. just click ok
2. Copy the current path of location folder which all your mods in that folder. let say i save my jar mods at download. so, just copy the location and paste it.
3. Copy your .minecraft/mods location. if you are using tlaucher just find the mods folder in there. (Location the original and crack version might be different)
4. Choose your mods to copy and click ok
